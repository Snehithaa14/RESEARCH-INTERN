# Nash Equilibrium :

Definition:
A situation where no player can improve their outcome by changing strategy alone.

Example:
Attacker:
- Attack
- Not Attack

Defender:
- Defend
- Not Defend

If both choose their best stable strategy, the system reaches Nash Equilibrium.

Application:
Used in cybersecurity and network decision-making.


# Stackelberg Game

Definition:
A leader-follower game.

Leader:
Defender

Follower:
Attacker

Process:
1. Defender chooses strategy.
2. Attacker observes.
3. Attacker responds.

Application:
Useful in network security and resource allocation.
